# reviews/image/

Optionele foto's die bij een specifieke review horen. Verwijs ernaar via het
`image`-veld in het bijbehorende JSON-bestand in `../text/`, bijvoorbeeld:

```json
{ "review": "...", "image": "dak-jan.jpg" }
```

Laat het `image`-veld leeg (`""`) als er geen foto bij de review hoort.
